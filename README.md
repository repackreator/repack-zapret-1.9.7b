Как запустить Zapret: 
 1. На страничке с моим репаком нажимаем на зеленую кнопку " <> Code"
 2. Нажимаем "Download ZIP", у вас скачается файл с названием "repack-zapret-1.9.7b-main.zip"
 3. Открываем этот ZIP архив
 4. Заходим в папку "repack-zapret-1.9.7b-main"
 5. Заходим в архив "repack-zapret-1.9.7b.zip"
 6. Запускаем файл "service.bat" (service)
 7. Вам высветится окно "Сжатые ZIP-папки", внизу будет 3 кнопки, из них нажимаем "Извлечь все"
 8. Открывается окно "Распаковка сжатых ZIP-папок"
 9. Выбираем куда извлекутся файлы (советую на рабочий стол по пути: C:\Users\"Ваше имя"\Desktop\zapret-discord-youtube-1.9.7b)
 10. Файлы извлекуться, теперь запускаем "service.bat" (service)
 11. Вам высветится окно "Открыть файл - предупреждение системы безопасности", нажимаем "Запустить"
 12. Вам высветится окно "Разрешить этому приложению вносить изменения на вашем устройстве", нажимаем "Да"
 13. В строку "Select option (0-11):" мы вписываем "1" и нажимаем Enter
 14. Выбираем любую опцию (0-19), советую выбирать ALT10 (это число 10) потому-что она менее проблемная, и нажимаем Enter
 15. Что-бы выключить Zapret в строку "Select option (0-11)" мы вписываем "2" и нажимаем Enter
